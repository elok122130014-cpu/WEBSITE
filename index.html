Monitoring Turbin VENTALUX

VENTALUX merupakan sistem monitoring turbin ventilator berbasis Internet of Things (IoT) yang mampu memantau kecepatan angin, RPM turbin, suhu, tegangan generator, arus generator dan daya generator secara realtime melalui website.

Jelajahi Pelajari

Keunggulan VENTALUX

Sistem monitoring yang dirancang untuk membantu pengguna memantau performa turbin ventilator secara realtime.

Monitoring Realtime

Data sensor diperbarui secara langsung sehingga kondisi sistem dapat dipantau kapan saja.

Internet of Things

ESP32 mengirimkan data sensor ke website menggunakan jaringan internet.

Energi Terbarukan

Memanfaatkan energi angin melalui turbin ventilator sebagai sumber energi alternatif.

Cara Kerja Sistem

VENTALUX bekerja dengan mengumpulkan data dari sensor, kemudian mengirimkannya ke website untuk ditampilkan secara realtime.

1

Turbin Ventilator

Turbin ventilator berputar akibat hembusan angin dan menghasilkan putaran mekanik.

2

Sensor & ESP32

Sensor membaca RPM, suhu, tegangan, arus, kemudian ESP32 mengirim data ke server.

3

Dashboard Website

Data ditampilkan dalam bentuk dashboard monitoring sehingga pengguna dapat memantau kondisi sistem.
