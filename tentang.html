Tentang VENTALUX

VENTALUX merupakan sistem monitoring turbin ventilator berbasis Internet of Things (IoT) yang dikembangkan untuk memantau kondisi turbin dan keluaran generator secara realtime melalui website.

Apa itu VENTALUX?

VENTALUX merupakan proyek Capstone Teknik Elektro ITERA yang bertujuan memanfaatkan putaran turbin ventilator sebagai sumber energi alternatif sekaligus menyediakan sistem monitoring berbasis Internet of Things.

Tujuan

Mengembangkan sistem monitoring yang mampu menampilkan kondisi turbin secara realtime.

Inovasi

Menggabungkan turbin ventilator, generator DC, ESP32, dan website monitoring dalam satu sistem.

Manfaat

Membantu proses monitoring serta mendukung pemanfaatan energi terbarukan yang ramah lingkungan.

Komponen Sistem

VENTALUX terdiri dari beberapa komponen utama yang saling terhubung untuk melakukan proses monitoring.

1

Turbin Ventilator

Mengubah energi angin menjadi energi mekanik melalui putaran turbin.

2

Generator & ESP32

Generator menghasilkan energi listrik, sedangkan ESP32 membaca data sensor dan mengirimkannya ke website.

3

Website Monitoring

Menampilkan data sensor, grafik monitoring, dan histori data secara realtime.

Alur Kerja Sistem

Proses monitoring VENTALUX dari turbin hingga data tampil pada website.

🌬️ Angin

Angin memutar turbin ventilator sehingga menghasilkan putaran mekanik.

⚙️ Generator

Generator mengubah putaran mekanik menjadi energi listrik dan sensor membaca parameter sistem.

📡 ESP32

ESP32 mengirimkan seluruh data sensor ke server melalui jaringan internet.

💻 Website

Data ditampilkan pada dashboard monitoring sehingga dapat dipantau secara realtime.
